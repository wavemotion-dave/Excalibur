# Excalibur
Excalibur RPN Calculator

Designed as an HP-like calculator for Windows.
Win32-based and will run on any version of Windows from 98 up to Windows 11 and generally runs quite well unde Linux wine.

Version 2.00 and 2.01 (essentially the same with an updated email address) were the last legacy versions available circa 2000.

I'm in the process of reviving this for a new 3.0 release with improved window size/font and some new features. Souce code will be made available shortly.
